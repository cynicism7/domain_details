# 1、启动模型
打开一个终端，输入以下命令

```bash
cd "/home/qiuyuqi/桌面/new program"
source .venv-linux/bin/activate

vllm serve ~/models/Qwen2.5-3B-Instruct/ --gpu-memory-utilization 0.85
```

看到如下输出表示模型启动正常

```bash
INFO:     Started server process [626267]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

# 2、配置config
打开config.vllm.example.yaml，修改如下部分
```bash
path_mapping:
  physical_root: "/media/qiuyuqi/D/85000000解压好"
  logical_root: "85000000解压好"

literature_dirs:
  - "/media/qiuyuqi/D/85000000解压好"

```
physical_root和literature_dirs为在当前系统下文件夹的物理路径

logical_root为一个虚拟路径，用于move over程序，在后续程序只需保证这个路径往下和在这个程序完全相同就没有问题

# 3、启动程序
再打开一个终端，输入以下命令
```bash
cd "/home/qiuyuqi/桌面/new program"
source .venv-linux/bin/activate
python main.py --config config.vllm.example.yaml scan
```
